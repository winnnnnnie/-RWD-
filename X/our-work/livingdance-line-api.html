舞拾人生 | LINE 官方帳號系統串接

教室營運管理系統，LINE+CRM會員溝通好輕鬆!

專案背景

專為熟齡學習者打造的
全方位芭蕾舞蹈教室
舞拾人生

舞拾人生是全台首間專為熟齡學習者開設的芭蕾舞蹈教室。取名為『舞拾人生』，是取『重拾舞蹈』和『50』 諧音，以『熟齡學生的需求』為出發點，為了熟齡族群的安全，細心打造教室軟、硬體，教室特別墊高彈性木頭地板、鋪設專用軟墊，減少學員傷害的可能性，希望透過舞蹈給於熟齡族群優雅豐富的生活，開啟熟齡族人生精彩的下半場。

然而，才藝教室的管理最常面臨行政作業繁瑣、課程紙本紀錄不易管理、會員資料分散等煩惱，隨著事業逐步擴張，更難掌握內部資源的安排。而透過數位營運系統，舞拾人生將原本依賴紙本的報名與上課紀錄全面數位化，大幅提升行政人員的工作效率，且讓學員報名流程更加方便！

專案目標

貼近熟齡族需求的
數位服務設計

為了讓熟齡學員更直覺地使用數位工具進行報名，舞拾人生選擇用 LINE官方帳號作為數位報名平台，透過LINE API 技術串接開發的課程報名系統，不需要額外開發獨立 APP，大幅降低了開發與維護成本。

另外，對熟齡學員來說，LINE是日常上再熟悉不過的通訊軟體，手機一打開LINE，找到舞拾人生的官方帳號就能隨時報名課程、查看課程資訊，甚至能夠自行請課、補課，查閱自己的上課紀錄。如果針對課程內容有疑問，也能直接在LINE官方帳號傳送訊息詢問行政人員，這些方便使用的細節設計，讓熟齡族群得以持續保有上課的熱情。

客製功能

熟齡族友善報名介面

為了確保熟齡學員能夠輕鬆操作報名介面，報名系統的畫面設計地對熟齡族群更加友善，包括字體放大、操作指示更明確、簡潔易懂的選單設計，都是為了讓熟齡族群使用上閱讀更清晰，降低任何使用上的障礙。

後台營運管理系統

跟隨事業擴展的
聰明自動生成系統

一站式後台營運管理系統，整合了『學生、老師、教室場地』三方資訊。對管理者來說，每一堂課都可以設定日期、上課人數和補課人數上限，避免發生課程超約的狀況，全面掌握好課程的高品質。

跨據點後台管理

跨據點的資料
統一後台管理

當事業進行拓展時，後台直接拓展新的授課地點、新課程及老師，方便行政人員輕鬆管理新教室的擴展，讓人力與教室資源得到最佳的運用。

學員可以在LINE官方帳號上選擇跨點報名課程，資料統一彙整在同一個系統，而團隊透過查閱系統資料來了解新生、訂單增長的趨勢，更能掌握行銷的成果與方向，高效地營運與拓展事業。

數位營運系統

會員管理系統

舞拾人生的數位營運系統累積了學員們多年來的上課紀錄、課程偏好、健康狀況等數據，這些數據都是未來決策行銷方案的重要基礎，例如說透過分析學員的運動頻率、健康狀況，能開發設計更多適合熟齡族的課程、推薦學員更個人化的課程，進一步提升品牌黏著度和擴大客戶群。

專案設計的出發點是為才藝教室打造一個數位營運系統，透過LINE API 技術串接開發的課程報名系統，帶領才藝教室走向數位化的旅程。而開發之後所累積的資料數據，更是業者未來再行銷、發想行銷策略規劃的重要參考，幫助業者能更高效地決策和營運，持續陪伴更多熟齡學員在芭蕾的世界中舞動人生！
