喬登健康管理中心

全臺據點統一Line線上預約
顧客方便 管理更方便！

專案背景

美食與健康的平衡者
喬登健康管理中心

在台灣經營20多年擁有多家分店的喬登健康管理中心，堅持與客人以面對面的方式進行諮詢，不靠吃藥或外力減重，專注飲食管理計畫的設計及陪伴支持，協助客人在美食與健康之間取得完美的平衡點，讓他們能夠在攝取美食的同時也能健康瘦身。並藉由口耳相傳的方式，在瘦身界佔有一席之地。

前往數位化的路上

過去為傳統經營管理模式，從客人購買服務到諮詢預約都是紙本記錄、人工操作，面對行動網路應用普及，改變了消費大眾的生活習慣，更便利的諮詢模式提供與時俱進的服務內容已成為必須的改變。而數據行銷的發展更是直接與客人的行為有了直達性的串聯關係產生。

系統開發目標

減少管理鎖事
活用數據提升業績

數位化的幫手 – Line官方帳號 + API = 會員APP

利用Line 官方帳號近距黏住客人進而達成密切諮詢的服務目標，相較於開發自有APP所需要花費的成本、時間和維護難度，Line API 擁有自身公司維護、開發，相對降低許多成本。加上Line在台灣的高市佔率，可省下鼓勵下載的推廣成本，透過官方帳號的公開曝光管道，更能快速分享給親朋好友，觸及新客。

每個店點都可設定當值營養師及客人的預約行事曆，清楚掌握一周的工作預約。
VIP會員透過LINE 官方帳號就可進入系統進行諮詢預約。

客製功能

會員飲食狀況
系統隨時掌握

VIP會員專屬的飲食拍照上傳，記錄每日進食內容，讓營養師從系統就可隨時掌握會員的飲食狀況。

會員激勵功能

成功諮詢換好禮

設計報到積點機制鼓勵會員持續往目標邁進，每成功諮詢一次，就能夠得到點數用以換取禮物與各式折扣。

商品預購功能

線上預購機制
掌握第一手限定消息

以導客重複回店及服務會員為主要目的的線上預購機制，方便會員每次到店諮詢時可順手帶回預先訂購商品。結合推播功能通知取貨，掌握第一手限定商品消息。

MGM獎勵機制

好康逗相報
讓會員幫你快速漲粉

飲食健康管理的成果，經常藉由口耳相傳在朋友圈內相互推薦合適的管理中心，透過MGM (Members get members)獎勵機制，提供誘因吸引現有顧客真心推薦，邀請朋友加入會員，新用戶同步獲得獎勵，迅速增加有效顧客名單，擴展潛在顧客規模，找到對的人！
